Hướng dẫn khởi chạy backend

Cài đặt Python 3.10 hoặc 3.11 (nên dùng Python 3.11 để tương thích tốt nhất).

Sử dụng lệnh sau để cài đặt các thư viện cần thiết:

	pip install -r requirements.txt

Sau đó, chạy file backend.py để khởi chạy backend thành công:

	python backend.py

Để kiểm thử, bạn cài thêm thư viện requests:

	pip install requests

Sau đó chạy file test.py để kiểm tra hoạt động:

	python test.py
